# NAK Incentive Web App - Full Function Scope

ระบบนี้ออกแบบสำหรับเว็บแอปภาษาไทยที่ใช้ตรวจสอบข้อมูล Incentive / รายได้ / รายการหักของพนักงาน โดยรองรับมือถือ คอม แท็บเล็ต และ PWA

## ฟังก์ชันหลัก

- Login ด้วยรหัสพนักงาน + PIN
- ขอเปิดใช้งานครั้งแรกด้วยรหัสพนักงาน + วันเริ่มงาน
- Manager/Admin อนุมัติก่อนใช้งาน
- อุปกรณ์ที่เชื่อถือได้ สูงสุด 2 อุปกรณ์ต่อคน
- Staff เห็นเฉพาะข้อมูลตัวเอง
- Manager เห็นเฉพาะ HUB/Area/ทีม
- Admin เห็นทั้งหมด
- อัปโหลด Excel
- Mapping คอลัมน์
- รองรับคอลัมน์เปลี่ยน เพิ่ม ลบ สลับตำแหน่ง
- เก็บ Raw Data ทุกคอลัมน์
- รองรับหลายประเภทไฟล์: Incentive, เบี้ยขยัน, OT, Attendance, Bonus, ค่าปรับ, อื่น ๆ
- เก็บประวัติย้อนหลังรายเดือน
- Import ซ้ำเป็น Version ได้
- Audit Log ทุกการกระทำ
- แจ้งปัญหาข้อมูล
- Reset PIN / Reset Device / Unlock Account

## ข้อความสำคัญในแอป

> หมายเหตุ: ยอดที่แสดงเป็นข้อมูลจากไฟล์ Incentive เท่านั้น ยังไม่รวมรายได้จากเบี้ยขยัน และอาจยังไม่ใช่ยอดรายได้สุทธิทั้งหมดของพนักงาน

## กฎความปลอดภัย

- ห้ามเก็บ PIN ตรง ๆ ต้องเก็บ hash เท่านั้น
- PIN ผิดเกิน 5 ครั้ง ล็อกบัญชี
- อุปกรณ์ใหม่ต้องขออนุมัติ
- API ทุกตัวตรวจ Role
- Firebase Admin SDK ใช้เฉพาะฝั่ง Server
- ทุกการดู แก้ ดาวน์โหลด อัปโหลด อนุมัติ ต้องมี Audit Log
